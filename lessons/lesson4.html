บทที่ 4: การคำนวณและนิพจน์ (Operators & Expressions)

นิพจน์ (Expression) คือการนำตัวแปรและค่าคงที่มาประมวลผลร่วมกับตัวดำเนินการ (Operators) เพื่อให้ได้ผลลัพธ์ใหม่ ใน Python มีตัวดำเนินการที่หลากหลายซึ่งช่วยให้การคำนวณทางคณิตศาสตร์และตรรกศาสตร์ทำได้ง่ายขึ้น

4.1 ตัวดำเนินการทางคณิตศาสตร์ (Arithmetic Operators)

// Floor Division: หารเอาส่วน (ตัดเศษทิ้ง)
% Modulus: หารเอาเศษ
** Exponentiation: การยกกำลัง
/ Division: การหาร (ได้ float เสมอ)

4.2 ลำดับความสำคัญ (Operator Precedence)

เมื่อมีตัวดำเนินการหลายตัวในบรรทัดเดียว Python จะเรียงลำดับตามกฎ PEMDAS จากความสำคัญสูงสุดไปต่ำสุด:

1. ( ) วงเล็บ
➜
2. ** ยกกำลัง
➜
3. * / // % คูณ/หาร
➜
4. + - บวก/ลบ
  • Parentheses (): มีความสำคัญสูงสุดเสมอ ใช้เพื่อบังคับทิศทางสมการ
  • Exponents **: คำนวณก่อนการคูณหรือหาร
  • Multiplication/Division: หากอยู่ในระดับเดียวกันจะทำจาก ซ้ายไปขวา

4.3 ตัวดำเนินการทางตรรกศาสตร์ (Logical Operators)

ใช้สำหรับการเปรียบเทียบค่า ซึ่งจะให้ผลลัพธ์เป็น True หรือ False เท่านั้น:

  • Comparison: == (เท่ากับ), != (ไม่เท่ากับ), >, <
  • Logical: and (และ), or (หรือ), not (ไม่)
