บทที่ 7: ฟังก์ชันและโมดูล (Functions & Modules)

เมื่อโปรแกรมมีความซับซ้อนมากขึ้น การแบ่งโค้ดออกเป็นส่วนย่อยๆ ที่ทำงานเฉพาะอย่าง (Modularization) จะช่วยให้เราไม่ต้องเขียนโค้ดซ้ำ และหาข้อผิดพลาดได้ง่ายขึ้นมากค่ะ

7.1 การนิยามฟังก์ชัน (Defining Functions)

ฟังก์ชันคือชุดคำสั่งที่สามารถนำกลับมาใช้ซ้ำได้ โดยใช้คำสำคัญ def และสามารถส่งค่ากลับได้ด้วย return

def calculate_vat(price):
    return price * 0.07

7.2 ประเภทของ Argument

ความยืดหยุ่นของ Python ในการส่งค่า:

  • Positional Arguments: ส่งตามลำดับ (เช่น add(5, 10))
  • Default Parameters: กำหนดค่าเริ่มต้น (เช่น def greet(msg="Hello"))

7.3 ขอบเขตของตัวแปร (Scope)

ตัวแปรที่ประกาศในฟังก์ชันเป็น Local Scope (เรียกใช้ข้างนอกไม่ได้) ส่วนตัวแปรที่ประกาศข้างนอกเป็น Global Scope
