บทที่ 2: การเตรียมสภาพแวดล้อมและการใช้เครื่องมือ

ก่อนที่เราจะเริ่มเขียนโปรแกรม พื้นฐานที่สำคัญที่สุดคือการตั้งค่าเครื่องมือให้พร้อมใช้งานเสมือนนักพัฒนามืออาชีพ บทนี้จะสอนขั้นตอนการติดตั้งและหัวใจสำคัญของการจัดการระบบ

2.1 การติดตั้ง Python และความสำคัญของ PATH

ในขั้นตอนการติดตั้ง Python บน Windows สิ่งที่สำคัญที่สุดคือการทำเครื่องหมายที่ "Add Python to PATH" ระบบ PATH คือตัวแปรของระบบปฏิบัติการที่บอกให้เครื่องรู้ว่าไฟล์รันโปรแกรมของ Python อยู่ที่ไหน

2.2 Visual Studio Code: Editor อันดับ 1

Visual Studio Code (VS Code) เป็นโปรแกรมแก้ไขโค้ดที่ได้รับความนิยมสูงสุดในโลกปัจจุบัน เนื่องจากความเบาและความสามารถในการขยาย (Extensions)

2.3 หัวใจสำคัญ: Virtual Environments (venv)

นักพัฒนาจึงนิยมใช้ Virtual Environment เพื่อสร้างพื้นที่จำลองแยกแต่ละโปรเจกต์ออกจากกัน เพื่อป้องกันปัญหา Library เวอร์ชันตีกัน

1. Create venv
(สร้างโฟลเดอร์จำลอง)
➜
2. Activate
(เปิดการใช้งานระบบ)
➜
3. Install Library
(ติดตั้งผ่าน pip)
➜
4. Coding!
(เริ่มเขียนโปรแกรม)

คำสั่งที่ควรรู้:

# 1. สร้างสภาพแวดล้อมจำลอง
python -m venv venv_name

# 2. การเปิดใช้งาน (Activate) บน Windows
.\venv_name\Scripts\activate

2.4 การจัดการแพ็คเกจด้วย PIP

pip (Package Installer for Python) คือเครื่องมือมาตรฐานในการดาวน์โหลดและจัดการ Library ภายนอกจาก PyPI

check_setup.py
