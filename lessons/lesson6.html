บทที่ 6: การวนซ้ำและการทำซ้ำ (Iteration)

ในโลกของการเขียนโปรแกรม การทำซ้ำงานเดิมๆ คือสิ่งที่คอมพิวเตอร์ทำได้ดีกว่ามนุษย์ Python จึงเตรียมเครื่องมือที่ทรงพลังในการจัดการกับข้อมูลจำนวนมากผ่านการวนซ้ำ (Loops)

6.1 For Loop: การวนซ้ำตามจำนวนรอบที่แน่นอน

for loop ใน Python มักใช้ร่วมกับออบเจกต์ที่วนรอบได้ (Iterables) เช่น List หรือการใช้ฟังก์ชัน range()

[Image of Python for loop diagram]
for i in range(5):
    print(f"รอบที่ {i}")

6.2 While Loop: การวนซ้ำตามเงื่อนไข

while จะทำงานไปเรื่อยๆ จนกว่าเงื่อนไขที่กำหนดจะเป็นเท็จ (False)

[Image of Python while loop flowchart]

6.3 เทคนิคขั้นสูง: List Comprehension

ฟีเจอร์เด็ดของ Python ที่ช่วยสร้าง List ในบรรทัดเดียว:

squares = [x**2 for x in range(10)]

6.4 การควบคุม Loop (Break & Continue)

  • break: สั่งให้หยุดการวนซ้ำทันที
  • continue: ข้ามโค้ดส่วนที่เหลือในรอบนั้นแล้วไปเริ่มรอบใหม่
