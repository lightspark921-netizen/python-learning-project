บทที่ 3: ตัวแปรและชนิดข้อมูล (Variables & Data Types)

ในโลกของโปรแกรมมิ่ง ตัวแปร (Variable) เปรียบเสมือน "กล่อง" ที่เราใช้เก็บข้อมูลไว้ในหน่วยความจำ (RAM) เพื่อนำมาประมวลผลในภายหลัง Python มีความพิเศษกว่าภาษาอื่นตรงที่เป็นระบบ Dynamic Typing

3.1 Dynamic Typing และกระบวนการจัดเก็บ

ใน Python เมื่อเราประกาศตัวแปร ระบบจะสร้าง Object ขึ้นมาในหน่วยความจำ และให้ชื่อตัวแปรนั้นชี้ (Point) ไปที่ตำแหน่งข้อมูลดังกล่าว ดังแผนภาพ:

[ แผนภาพการจองพื้นที่ในหน่วยความจำ ]

Variable: name"Umi"
Variable: age25
Variable: pi3.14

เราสามารถประกาศ x = 5 ได้ทันที และยังสามารถเปลี่ยนเป็นข้อความได้ในภายหลัง ระบบจะจัดการเนื้อที่ในหน่วยความจำให้โดยอัตโนมัติ

3.2 ชนิดข้อมูลพื้นฐาน (Standard Data Types)

ตารางสรุปชนิดข้อมูลที่พบบ่อยที่สุดในการเขียนโปรแกรม:

Typeชื่อเรียกตัวอย่าง
intเลขจำนวนเต็ม10, -5
floatเลขทศนิยม3.14
strข้อความ (String)"Python"
boolค่าตรรกศาสตร์True, False

3.3 โครงสร้างข้อมูลแบบชุด (Collection Types)

นอกจากตัวแปรเดี่ยว Python ยังมีโครงสร้างข้อมูลที่ซับซ้อนขึ้น:

  • List [ ]: เก็บข้อมูลเรียงต่อกัน แก้ไขค่าได้ (Mutable)
  • Tuple ( ): เก็บข้อมูลเรียงต่อกัน แต่แก้ไขค่าไม่ได้ (Immutable)
  • Dictionary {k:v}: เก็บข้อมูลแบบคู่ Key-Value เพื่อความเร็วในการค้นหา
