บทที่ 5: การควบคุมทิศทางโปรแกรม (Control Flow)

การควบคุมทิศทางโปรแกรมคือการสร้างเงื่อนไขเพื่อให้คอมพิวเตอร์เลือกทำงานตามสถานการณ์ที่เกิดขึ้นจริง ทำให้โปรแกรมมีความฉลาดและสามารถตอบโต้กับผู้ใช้ได้อย่างถูกต้อง

5.1 โครงสร้าง If-Elif-Else

เป็นการตรวจสอบเงื่อนไขแบบเรียงลำดับ หากเงื่อนไขใน if เป็นเท็จ โปรแกรมจะขยับไปเช็คใน elif (Else If) และหากไม่ตรงกับเงื่อนไขใดเลย จะมาตกที่ else

[Image of Python If-Else flowchart]
กฎเหล็กของ Python: Indentation (การย่อหน้า)
Python ไม่ใช้ปีกกา {} แต่ใช้การย่อหน้าเพื่อบอกขอบเขตของโค้ด หากย่อหน้าไม่เท่ากัน โปรแกรมจะเกิดข้อผิดพลาด IndentationError ทันที

5.2 Logical Operators ในเงื่อนไข

  • and: ต้องจริงทุกเงื่อนไข
  • or: จริงเพียงเงื่อนไขเดียวก็ได้
  • not: กลับค่าจากจริงเป็นเท็จ และเท็จเป็นจริง

5.3 Match-Case (ฟีเจอร์ใหม่!)

ใน Python 3.10 ขึ้นไป ได้มีการเพิ่ม match-case เข้ามาเพื่อใช้แทนการใช้ if-elif หลายๆ ชั้น ช่วยให้โค้ดอ่านง่ายขึ้น
